Introduktion

Syftet med denna guide är att ge en introduktion till kärnan, de väsentliga delarna, i programmeringsspråket JavaScript, så som en programmerare tar sig an ett nytt programmeringsspråk.
### Versioner {#versioner} JavaScript uppdateras flitigt och vid varje uppdatering tillkommer det funktionalitet. Webbläsarna, som renderar JavaScripten, måste då uppdateras för att kunna hantera den nya funktionaliteten. De två "stora" uppdateringarna var ES5 (ca 2009) och ES6 (ca 2015). Den senaste, i skrivande stund, är ES2020 vilken kom i juni 2020. I den här kursen kommer vi ha ett fokus på grunderna i språket och hur det fungerar på klientsidan. Det finns exempel som bygger på ES5 och de som bygger på ES6. Vi har valt att inte dra det längre, utan vi håller oss däromkring. Det kan såklart finnas konstruktioner i senare versioner som går att använda, vilket uppmuntras om stödet finns i de vanliga webbläsarna.
